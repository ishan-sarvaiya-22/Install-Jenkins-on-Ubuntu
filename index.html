Steps for Installing Jenkins on Ubuntu

What is Jenkins?

Jenkins is an open-source automation server that helps automate software development processes, including building, testing, and deploying applications. It provides a wide range of plugins and integrations that enable continuous integration and delivery (CI/CD) pipelines, facilitating faster and more efficient software development.

Step 1: Updating the Package Lists

To begin, update the package lists on your Ubuntu server. This ensures that the system has the latest information about available packages and their versions.

sudo apt-get update

Step 2: Installing Java Runtime Environment (JRE)

Jenkins requires Java to run. Install the default Java Runtime Environment (JRE) package using the following command:

sudo apt-get install default-jre

Step 3: Adding the Jenkins Repository Key

In this step, you will add the Jenkins repository key to ensure the authenticity of the Jenkins package that you are about to install.

wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -

Step 4: Adding the Jenkins Repository

Add the Jenkins repository to the package sources list to enable the system to fetch Jenkins packages during installation.

sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'

Step 5: Updating the Package Lists (Again)

Update the package lists again to include the newly added Jenkins repository.

sudo apt-get update

Step 6: Installing Jenkins

Now, it's time to install Jenkins on your Ubuntu server using the following command:

sudo apt-get install jenkins

Step 7: Starting Jenkins

After the installation is complete, you can start the Jenkins service with the following command:

sudo systemctl start jenkins

Step 8: Enabling Jenkins on System Startup

To ensure that Jenkins starts automatically when the server boots up, use the following command:

sudo systemctl enable jenkins

Step 9: Checking Jenkins Service Status

Verify the status of the Jenkins service to ensure it is running correctly:

sudo systemctl status jenkins

Step 10: Accessing Jenkins Web Interface

Open a web browser and navigate to http://your_server_ip:8080/. This will allow you to access the Jenkins web interface where you can configure and manage your Jenkins instance.

Step 11: Obtaining the Initial Administrator Password

To complete the Jenkins setup, you need to retrieve the initial administrator password. Execute the following command to obtain it:

sudo cat /var/lib/jenkins/secrets/initialAdminPassword
